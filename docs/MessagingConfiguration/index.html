is​Multiline​Response​Options​Enabled 

@objc public var isMultilineResponseOptionsEnabled = false

Flag to enable whether the user response options will span across multiple lines, or just have one line.

  • false: ResponseOptions appear in a UICollectionView which scrolls horizontally if the contentSize.width is greater than the collectionView'sframe.width

  • true: ResponseOptions appear in a UICollectionView which expands vertically to fit all the content on screen

eg. AnswerBot response options or Chat's pre-chat form department selection.
