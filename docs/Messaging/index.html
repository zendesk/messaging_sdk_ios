Class Messaging 

@objc(ZDKMessaging)
public final class Messaging: NSObject, NotificationCenterObserver

The Messaging class coordinates Zendesk's product UI engines, to allow them to work seamlessly in a single unified messaging interface.

A UIViewController can be built, once the Messaging singleton is initialized successfully. This will allow an end-user to start their customer support experience. Messaging contains a list of Engines which map to different Zendesk SDKs like AnswerBot, Chat, and Support.
